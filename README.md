# 🌟 IponTubo - Yield & Dividend Calculator

IponTubo is a premium, interactive financial planner and investment companion built to help you calculate, visualize, and forecast returns on Retail Treasury Bonds (RTBs) and Pag-IBIG MP2 savings in the Philippines.

## ✨ Features

- 🏛️ **Retail Treasury Bonds (RTB)**: Calculate net quarterly payouts (including equivalent monthly metrics), real returns after inflation, and total interest earned.
- 🇵🇭 **Pag-IBIG MP2**: Simulate tax-free compound growth or annual payouts. See exactly how much you can earn based on your contributions and expected dividend rate.
- 🎯 **Goal-Seek Calculator**: Have a financial target in mind? Reverse-engineer your investments for both MP2, Treasury Bonds, or custom instruments to find out exactly how much you need to deposit.
- 📉 **Inflation Adjuster**: Turn on the inflation adjuster to view the "Real Return" of your investment over time, taking expected inflation rates into account.
- 💾 **Save & Export**: Save multiple investment scenarios and export your entire portfolio of saved calculations into a beautifully formatted PDF.
- 🌕 **Dark/Light Themes**: Intuitive and modern UI design with a seamless dark and light mode toggle.
- 📚 **Knowledge Base & Walkthroughs**: Need help? The built-in interactive tour, simulated video guides, and FAQs section contain everything you need to start investing properly.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or pnpm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
